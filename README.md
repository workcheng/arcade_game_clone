这是一个街机游戏克隆项目

玩法
-
 - 在浏览器中打开index.html
 - 通过移动上下左右键来控制玩家的移动
 - 不能碰到平行移动的昆虫敌人，如果碰到将会重新从原点开始
 - 当玩家通过河对岸，则赢得胜利！
 
 文件结构
 -
````

-
 -css
  -style.css     样式
 -images         图片资源
 -js
  -app.js        定义Player和Enemy相关操作
  -engine.js     游戏引擎
  -resources.js  图片加载工具
 index.html      入口文件
 README.md       说明文件
````
